Schrecksekunde

Die Aufgabenstellung bestand daraus, das typische Verhalten eines Tieres mithilfe von Polygonen darzustellen.
Hierbei sollte das Tiermodell so reduziert wie möglich sein. In der Animation „Schrecksekunde“ wird das Reh und dessen Starrezustand im Scheinwerferlicht von Autos thematisiert. Besonderheit der Animation ist, dass sich die Handlung rückwärts abspielt: ein Autounfall, welcher mit dem angefahrenen Reh startet und mit dem gesunden, wegspringenden Reh endet.
